== Description ==

Organize links to the latest posts on a story below each post

Simply add the shortcode into the post or page of your choice, specifying a tag that groups all the related posts together. E.g.;
[organize_stories tag="Copenhagen27"]

Documentation is available via [Konfeldt.com](http://konfeldt.com/)

== Installation ==

1. Upload `organize_stories.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the `[organize_stories tag="your-tag"]` shortcode in your posts
